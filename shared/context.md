# 🎯 Contexte Partagé — Projet Renoir

## Mission
Construire une plateforme d'analytics du marché de l'art combinant :
- Trading bancaire appliqué à l'art
- Détection d'artistes émergents
- Recommandation de tableaux
- Copilote IA pour événements artistiques

- **Communication** : Telegram + ce repo GitHub
- **Hébergement** : VPS Hostinger (Docker)

## Données Marché Clés (Q1 2026)
- Marché global : 59,6 Mds $ (+4%)
- Sweet spot : segment 5K-50K € (artistes émergents)
- Tendances : surréalisme, art physique, mixed-media/collage
- 72% des collectionneurs intéressés par les émergents
